## 프로젝트 개요

AI와 함께한 모든 프로젝트와 작업 일지를 기록하는 개인 포트폴리오 웹사이트입니다.
**프로젝트 갤러리 + 블로그** 하이브리드 형태로, MDX 파일 기반의 정적 사이트입니다.

## 주요 기능

- **프로젝트 갤러리**: 카드 그리드 형태의 프로젝트 목록 + 상세 페이지
- **블로그**: 작업 일지와 기술 포스트
- **다크모드**: next-themes 기반 시스템 연동
- **코드 하이라이팅**: rehype-pretty-code + shiki
- **SEO**: 페이지별 메타데이터, Open Graph, sitemap

## AI 활용

이 프로젝트 자체가 **Claude Code와의 협업 결과물**입니다.

- 프로젝트 구조 설계: 디렉토리 구조, 기술 스택 선정
- 전체 코드 구현: 레이아웃, 컴포넌트, 콘텐츠 시스템
- 배포 자동화: GitHub Actions 워크플로우 설정

## 기술 스택 선정 이유

| 기술 | 이유 |
|------|------|
| Next.js 15 | App Router + 정적 export 지원 |
| MDX | 마크다운에 컴포넌트 삽입 가능 |
| Tailwind CSS v4 | 빠른 스타일링 + 다크모드 |
| GitHub Pages | 무료 호스팅 + CI/CD |
