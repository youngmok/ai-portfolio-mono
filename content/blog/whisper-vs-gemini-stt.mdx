## 배경

[이전 글](/ai-portfolio/blog/building-korean-stt/)에서 Faster-Whisper 기반 한국어 STT 시스템을 만들었습니다.
당시에는 **CPU(int8) 모드**로 동작했는데, CUDA 환경을 구축하고 **GPU(float16) 모드**로 전환한 뒤 성능이 어떻게 달라졌는지, 그리고 Google Gemini STT와 비교하면 어떤 차이가 있는지 실측했습니다.

## CUDA 환경 구축

RTX 3060 Ti(8GB VRAM)에 NVIDIA 드라이버(560.94)는 설치되어 있었지만, CUDA 런타임 라이브러리가 없어 GPU를 사용하지 못하고 있었습니다.

전체 CUDA Toolkit을 설치할 필요 없이 pip 패키지 2개로 해결했습니다.

```bash
pip install nvidia-cublas-cu12 nvidia-cudnn-cu12
```

Windows에서는 DLL 경로를 Python이 자동으로 찾지 못하는 문제가 있어, 모든 STT 모듈에 다음 코드를 추가했습니다.

```python
import os
import importlib.util

def _setup_cuda_dll():
    """Windows 환경에서 NVIDIA CUDA DLL 경로를 등록한다."""
    if os.name != "nt":
        return
    try:
        spec = importlib.util.find_spec("nvidia.cublas")
        if spec and spec.submodule_search_locations:
            cublas_bin = os.path.join(spec.submodule_search_locations[0], "bin")
            if os.path.isdir(cublas_bin):
                os.add_dll_directory(cublas_bin)
        spec = importlib.util.find_spec("nvidia.cudnn")
        if spec and spec.submodule_search_locations:
            cudnn_bin = os.path.join(spec.submodule_search_locations[0], "bin")
            if os.path.isdir(cudnn_bin):
                os.add_dll_directory(cudnn_bin)
    except Exception:
        pass

_setup_cuda_dll()
```

핵심은 `os.add_dll_directory()`입니다. Windows의 DLL 검색 경로에 pip으로 설치된 NVIDIA 라이브러리 위치를 직접 등록합니다.

## GPU vs CPU 속도 비교

10.9초 테스트 음성으로 같은 모델(large-v3)을 CPU와 GPU로 비교했습니다.

| 항목 | CPU (int8) | GPU (float16) | 차이 |
|------|-----------|---------------|------|
| **처리 시간** | 94.2초 | 2.2초 | **43배 빠름** |
| **배속** | 0.1x (실시간보다 느림) | 5.1x | - |
| **모델 로딩** | 13.5초 | 10.3초 | 1.3배 빠름 |

CPU에서는 10초 음성을 인식하는 데 94초가 걸렸습니다. **실시간보다 9배 느린** 수준입니다.
GPU 전환 후에는 2.2초로, **실시간보다 5배 빠르게** 처리됩니다.

## 22분 실전 음성 테스트

실제 22.7분(1363.7초) 한국어 음성을 Faster-Whisper(GPU)와 Google Gemini로 각각 인식했습니다.

### 처리 속도

| 항목 | Faster-Whisper (GPU) | Google Gemini |
|------|---------------------|---------------|
| **처리 시간** | 261.5초 (4분 22초) | 클라우드 처리 |
| **배속** | 5.2x | - |
| **세그먼트 수** | 564개 | 122개 |
| **장치** | RTX 3060 Ti (로컬) | Google Cloud |

Whisper는 22분 음성을 **4분 22초**에 처리했습니다. GPU 없이 CPU로 했다면 약 3시간 이상 걸렸을 겁니다.

### 세그먼트 분할 방식

Whisper는 2~5초 단위로 세밀하게 나누고, Gemini는 5~30초 단위로 큰 덩어리를 만듭니다.

```
Whisper (564개):
[0.0s → 4.7s]  얼마 전에 이영돈 PD님의 부정선거 그 실체를 밝힌다
[4.7s → 6.8s]  라고 하는 다큐를 혹시 보신 적 있으신가요?

Gemini (122개):
[0:00-0:05]  얼마 전에 이영돈 PD님의 부정선거 그 실체를 밝힌다라고 하는 답을 혹시 보신 적 있으신가요?
```

자막 생성 용도에는 Whisper가, 전체 내용 요약에는 Gemini가 읽기 편합니다.

## 인식 정확도 비교

### 인명/고유명사

정치 관련 한국어 콘텐츠라 인명이 자주 등장했습니다. 여기서 큰 차이가 났습니다.

| 원래 발화 | Whisper | Gemini |
|-----------|---------|--------|
| 윤석열 | **윤석열** | 윤성열 |
| 민경욱 | **민경욱** | 민경국 |
| 이낙연 | **이낙연** | 인학연 |
| 한동훈 | **한동훈** | 한동은 |
| 광주광역시 광산구 | **광주광역시 광산구** | 광주광의씨 광상구 |

Whisper가 한국어 고유명사 인식에서 **압도적으로 정확**합니다.

### 전문 용어 (가장 큰 차이)

이 영상은 통계학 개념을 설명하는 내용이어서 전문 용어가 핵심이었습니다.

| 용어 | Whisper | Gemini |
|------|---------|--------|
| **무작위/무작위성** | 일관되게 정확 | **거의 전부 "무작기/무작기성"** |
| 신뢰구간 | **신뢰구간** | 실뢰 구간 |
| 대수의 법칙 | **대수의 법칙** | 재수의 법칙 |
| 표본 | **표본** | 표보/표분/표반 (간헐 오류) |
| Z값 | **Z값** | z류값 |

"무작위성"은 영상의 **핵심 개념**인데, Gemini가 거의 모든 구간에서 "무작기성"으로 잘못 인식한 것은 치명적입니다. 이 오류 하나만으로도 전체 내용의 맥락을 이해하기 어려워집니다.

### Gemini가 더 나은 경우

모든 면에서 Whisper가 우수한 것은 아닙니다.

| 구간 | Whisper | Gemini |
|------|---------|--------|
| "고상한 말로" | 고사감말로 | **고상한 말로** |
| "투표함에서" | 투표하면서 | **투표 함에서** |
| 득표율 59% | 99% (오류) | **90 59% (부분 정확)** |

또한 Whisper에서는 간헐적으로 **hallucination**(환각)이 발생했습니다. 다른 시간대의 문장이 엉뚱한 위치에 중복 출력되는 현상입니다. Gemini에서는 이런 현상이 관찰되지 않았습니다.

## 종합 평가

| 평가 항목 | Whisper | Gemini |
|-----------|---------|--------|
| 인명 정확도 | **9/10** | 4/10 |
| 전문 용어 정확도 | **9/10** | 4/10 |
| 숫자/퍼센트 | **8/10** | 7/10 |
| 일반 텍스트 | **8.5/10** | 7/10 |
| 세그먼트 품질 | **9/10** | 6/10 |
| 환각 없음 | 7/10 | **8/10** |
| **종합** | **8.4/10** | **6.1/10** |

### Whisper 강점
- 인명, 지명, 전문 용어 인식이 정확
- 세밀한 타임스탬프 (자막 생성에 적합)
- 로컬 실행 (비용 없음, 프라이버시 보장)

### Gemini 강점
- 문단 단위로 읽기 편한 출력
- 환각(hallucination) 현상 없음
- 일부 자연스러운 표현 처리

## 정리

| 항목 | CPU (이전) | GPU (현재) | 개선 |
|------|-----------|-----------|------|
| 10초 음성 | 94.2초 | 2.2초 | **43배** |
| 22분 음성 | ~3시간 (추정) | 4분 22초 | **~40배** |
| 배속 | 0.1x | 5.2x | 실시간 5배 |

GPU 전환은 pip 패키지 2개 설치와 DLL 경로 등록 코드 추가만으로 완료했습니다. CUDA Toolkit 전체를 설치할 필요 없이, `nvidia-cublas-cu12`와 `nvidia-cudnn-cu12`만으로 충분합니다.

한국어 전문 콘텐츠의 STT 정확도는 **Faster-Whisper large-v3가 Gemini보다 확실히 우수**합니다. 특히 인명과 전문 용어에서 큰 차이를 보이며, 로컬 GPU에서 무료로 사용할 수 있다는 점이 최대 장점입니다.
